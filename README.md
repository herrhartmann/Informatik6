# Informatikkurs für die 6. Klasse

Willkommen zum Onlinekurs Informatik für die 6. Klasse! Dieser Kurs behandelt die grundlegenden Konzepte der Informatik. Wir werden uns mit Informationen, Daten, Codierung, Programmierung und künstlicher Intelligenz beschäftigen.

---

## 1. Informationen und Daten

### 1.1 Computer als Informatiksysteme

> **Definition:** Ein Informatiksystem ist ein System, das zur Speicherung, Verarbeitung und Übertragung von Informationen und Daten genutzt wird.

Ein Computer ist ein solches System. Er besteht aus Hardware (den physischen Bestandteilen) und Software (den Programmen, die auf dem Computer laufen). Gemeinsam ermöglichen sie das Verarbeiten und Speichern von Daten.

#### 📝 **Aufgabe 1.1:**  
Finde heraus, welche Hardwarekomponenten in deinem Computer verbaut sind. Liste sie auf und erkläre, welche Funktion sie jeweils haben.

### 1.2 Verwaltung von Dateien

> **Info:** Dateien sind digitale Sammlungen von Informationen. Wir speichern Texte, Bilder, Musik und vieles mehr in Dateien.

Dateien werden in einem Dateisystem organisiert, das dem Computer ermöglicht, schnell auf sie zuzugreifen. Ordner und Unterordner helfen uns, Dateien strukturiert abzulegen.

#### 📝 **Aufgabe 1.2:**  
Erstelle eine Ordnerstruktur auf deinem Computer und speichere dort eine Datei. Benenne die Ordner so, dass du leicht finden kannst, was du suchst.

### 1.3 Dateiformate

Dateien können unterschiedliche Formate haben, z. B. `.txt` für Textdateien, `.jpg` für Bilder oder `.mp3` für Musik. Das Dateiformat bestimmt, wie die Daten in der Datei organisiert sind und mit welchem Programm sie geöffnet werden können.

#### 📝 **Aufgabe 1.3:**  
Finde und notiere drei verschiedene Dateiformate und beschreibe, wofür sie verwendet werden.

### 1.4 Das Internet

Das Internet ist ein globales Netzwerk, das Computer auf der ganzen Welt miteinander verbindet. Es ermöglicht den Austausch von Informationen in Form von Webseiten, E-Mails und mehr.

#### 📝 **Aufgabe 1.4:**  
Recherchiere, wie das Internet funktioniert und schreibe eine kurze Zusammenfassung in eigenen Worten.

### 1.5 Das EVA-Prinzip

Das EVA-Prinzip steht für **Eingabe - Verarbeitung - Ausgabe**. Ein Computer nimmt Daten (Eingabe) auf, verarbeitet sie und gibt dann das Ergebnis (Ausgabe) zurück. Zum Beispiel, wenn du eine Suche im Internet durchführst.

#### 📝 **Aufgabe 1.5:**  
Beschreibe einen Alltagsgegenstand, der das EVA-Prinzip verwendet.

## 2. Daten und Codierung

### 2.1 Codes und Daten

In der Informatik nutzen wir Codes, um Daten darzustellen. Ein Code ist eine Zuordnung von Zeichen oder Symbolen, die Informationen repräsentieren. Das binäre Zahlensystem ist ein wichtiger Code in der Informatik.

#### 📝 **Aufgabe 2.1:**  
Warum nutzen Computer das binäre System? Finde eine Erklärung und stelle sie in eigenen Worten dar.

### 2.2 Winkeralphabet, Morsecode, Caesar-Verschlüsselung

- **Winkeralphabet:** Mit Fahnen können Nachrichten über große Entfernungen übermittelt werden.
- **Morsecode:** Ein Code aus Punkten und Strichen, der bei Funknachrichten genutzt wurde.
- **Caesar-Verschlüsselung:** Ein Verschlüsselungsverfahren, bei dem jeder Buchstabe im Alphabet um eine feste Anzahl von Stellen verschoben wird.

#### 📝 **Aufgabe 2.2:**  
Schreibe deinen Namen im Morsecode auf und verschlüssele ihn anschließend mit der Caesar-Verschlüsselung.

## 3. Programmieren I

### 3.1 Algorithmen im Alltag

Ein Algorithmus ist eine Schritt-für-Schritt-Anleitung, um ein Problem zu lösen. Algorithmen begegnen uns überall – z.B. bei einer Kochanleitung.

#### 📝 **Aufgabe 3.1:**  
Schreibe einen Algorithmus, um ein Butterbrot zu schmieren. Verwende dabei klare und logische Schritte.

### 3.2 Erste Schritte in Scratch

Scratch ist eine visuelle Programmiersprache, die ideal für den Einstieg ins Programmieren ist. In Scratch werden Befehle als Blöcke aneinandergereiht.

#### 📝 **Aufgabe 3.2:**  
Erstelle in Scratch ein Programm, in dem eine Figur sich auf dem Bildschirm bewegt und dabei ein Geräusch macht.

### 3.3 Ein Scratch-Projekt für Anfänger

In diesem Projekt wirst du ein einfaches Spiel erstellen. Zum Beispiel ein Fangspiel, in dem eine Figur ein Objekt einsammelt.

#### 📝 **Aufgabe 3.3:**  
Erstelle ein Fangspiel in Scratch. Die Figur soll sich steuern lassen und Punkte sammeln, wenn sie ein bestimmtes Objekt berührt.

## 4. Programmieren II

### 4.1 Entscheidungen

Entscheidungen in der Programmierung werden durch bedingte Anweisungen wie **wenn-dann** ermöglicht. So kann ein Programm auf verschiedene Eingaben unterschiedlich reagieren.

#### 📝 **Aufgabe 4.1:**  
Erstelle ein Scratch-Programm, das eine Entscheidung trifft – z. B. "Wenn die Figur an den Rand stößt, dann kehrt sie um."

### 4.2 Bedingte Wiederholungen

Bedingte Wiederholungen (Schleifen) führen Anweisungen wiederholt aus, solange eine bestimmte Bedingung erfüllt ist. 

#### 📝 **Aufgabe 4.2:**  
Programmiere in Scratch eine Schleife, in der eine Figur sich so lange bewegt, bis sie ein bestimmtes Objekt berührt.

### 4.3 Klassen und Objekte

In der objektorientierten Programmierung sind Objekte wie reale Dinge: Sie haben Eigenschaften und Methoden. Klassen sind Baupläne für Objekte.

#### 📝 **Aufgabe 4.3:**  
Stelle dir eine Katze als Objekt vor. Schreibe auf, welche Eigenschaften (z. B. Farbe) und Methoden (z. B. miauen) sie haben könnte.

## 5. Automatisierung und Künstliche Intelligenz

### 5.1 Was sind Automaten?

Automaten sind Systeme, die festgelegte Aufgaben selbstständig ausführen. Beispiele sind Ampelschaltungen oder Waschmaschinenprogramme.

#### 📝 **Aufgabe 5.1:**  
Nenne drei Automaten, die du im Alltag kennst, und beschreibe kurz, wie sie funktionieren.

### 5.2 Künstliche Intelligenz

Künstliche Intelligenz (KI) ist die Fähigkeit von Maschinen, Aufgaben auszuführen, die normalerweise menschliche Intelligenz erfordern – zum Beispiel Spracherkennung oder Bilderkennung.

#### 📝 **Aufgabe 5.2:**  
Recherchiere, was KI im Alltag macht. Finde drei Beispiele für KI-Anwendungen.

### 5.3 Maschinelles Lernen

Maschinelles Lernen ist ein Teilbereich der KI. Maschinen lernen aus Daten, um Muster zu erkennen und Vorhersagen zu treffen. Ein Beispiel ist die Bilderkennung.

#### 📝 **Aufgabe 5.3:**  
Stell dir vor, du bringst einer Maschine bei, zwischen Hunden und Katzen zu unterscheiden. Was müsste die Maschine „lernen“, um das zu schaffen?
